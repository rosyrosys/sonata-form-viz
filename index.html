Sonata Form Visualization

W. A. Mozart, Piano Sonata in C Major, K. 545, Mvt. I — a web-based learning tool that synchronizes score, audio, and formal analysis on a single timeline.

Sonata Form Visualization

The tool runs entirely in your browser. A Public Domain recording (Robin Alciatore, via Musopen / Wikimedia Commons) is bundled with each edition, so no additional setup is needed. Users who prefer a different performance can drop their own file at assets/mozart_k545_mvt1.mp3 (see each edition's assets/README.md). A WebAudio synth fallback plays tonic-pitched tones if the audio is unavailable.
