# Figures for paper Appendix A

The paper's Appendix A references three figures. Capture these from the
live demo at https://rosyrosys.github.io/sonata-form-viz/en/ (or the
local server at port 8772) before final submission.

## Capture instructions (Windows: Win+Shift+S → Snipping Tool)

### Figure A.1 — System overview at startup
**File:** `fig_a1_overview.png` (recommended 1440×900)
**State:** Load the page, do not press play. Capture the full viewport
showing: timeline at top, OSMD-rendered score below, control panel
(section jump buttons, calibration controls) on the side.

### Figure A.2 — Timeline close-up
**File:** `fig_a2_timeline.png` (recommended 1200×400)
**State:** Crop just the timeline strip. Make sure all three section
colors (exposition pastel-blue, development pastel-amber, recapitulation
pastel-green) and the theme accent bars (P / T / S / codetta) are
visible. The red playhead should be near the start.

### Figure A.3 — Recapitulation P with F-major overlay
**File:** `fig_a3_recap_p.png` (recommended 1440×900)
**State:** Click "Recapitulation (m. 42)" in the section-jump panel.
Wait for the score to scroll so the F-major P region is visible.
Capture the score area showing the green section overlay through the
recapitulation, with the now-playing badge showing "F major" in the
analysis pane.

## After capture

Place the PNG files directly in this `figures/` folder. The paper's
Appendix A captions already reference Figure A.1 / A.2 / A.3 by name.
For TISMIR final submission, also export as 300 DPI TIFF or vector
PDF if the production editor requests it.
